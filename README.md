# HappyHouse
공공데이터 API를 이용한 주택 실거래가 제공 웹

## Getting Started  

1. sql 파일들 실행
2. Spring Boot App Run
3. happyhouse를 vscode로 연다.
4. vscode 터미널에 npm install
5. npm install 
6. npm install --save jw-vue-pagination
7. npm run build
